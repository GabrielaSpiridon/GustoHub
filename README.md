GustoHub - Recipe Management SPA
GustoHub is a full-stack Single Page Application (SPA) designed for food enthusiasts to create, discover, and manage culinary recipes. This project demonstrates a robust implementation of the MEVN stack (modified with Firebase), focusing on real-time data, secure authentication, and a scalable NoSQL architecture.

Technical Stack
Frontend: Vue 3 (Composition API), Pinia (State Management), Vue Router.

Backend: Node.js, Express.js.

Database & Auth: Firebase Firestore (NoSQL), Firebase Authentication.

Styling: Responsive CSS / Tailwind CSS.

Data Generation: Faker.js for realistic seed data.
